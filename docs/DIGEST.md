## 5. The §1031 conflict

| Term sheet (LLC Day 1) | §1031 requirement |
|---|---|
| Kao gets **95% LLC membership** | Replacement must be **like-kind real property** |
| Promote inside JV | Partnership interest **not** real property ([§1031(a)(1)](https://www.law.cornell.edu/uscode/text/26/1031), [Reg. §1.1031(a)-3](https://www.law.cornell.edu/cfr/text/26/1.1031(a)-3)) |
| No QI/EAT in term sheet | Ground-up build needs **QI + EAT** ([Rev. Proc. 2000-37](https://www.irs.gov/pub/irs-drop/rp-00-37.pdf)) |
| Titan sells Belara; term sheet says Kao Trust | **Same taxpayer** through exchange ([Reg. §1.1031(k)-1(g)(4)](https://www.law.cornell.edu/cfr/text/26/1.1031(k)-1)) |
