# North Forsyth §1031 — Structure A

**Live site:** https://ethancow.github.io/North-Forsyth-1031-Diagrams/

Tabbed explainer for the Belara Apartments → North Forsyth Commerce Center §1031 exchange,
built around the recommended legal structure: **Structure A — "Exchange First, Then Combine."**
Kao exchanges Belara into **100% fee title**, then — after a genuine seasoning period — forms
the promoted Hanover joint venture by a tax-free **§721** contribution. This replaces the prior
development-TIC + promote-as-fee content, which a tax review concluded was likely to **fail** §1031.

This is a **conditional, more-likely-than-not** position (not "clean," and nothing here rises to
"will"); its strength rises with seasoning. A simpler, stronger fallback (**Structure B**) is included.

## How it's built

`index.html` fetches `North_Forsyth_1031_Diagrams.md`, splits it on `<!-- TAB:name -->` markers,
and renders Markdown (marked.js) + diagrams (mermaid). The source of truth lives outside this repo
as `North_Forsyth_1031_StructureA.md`; this markdown is generated from it.
